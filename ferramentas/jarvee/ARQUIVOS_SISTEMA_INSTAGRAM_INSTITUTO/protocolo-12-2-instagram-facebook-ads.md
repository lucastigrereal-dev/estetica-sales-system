## 2. AD TYPES

**Feed Ads:** Appear in feed, single image/video, best CTR ($0.20-0.50/click)
**Story Ads:** 9:16 vertical, full screen, swipe up ($0.15-0.30)
**Reels Ads:** Between Reels, native format, cheapest ($0.10-0.25)
**Carousel Ads:** Multiple images/videos, high engagement ($0.30-0.60)

- Conversions (sales)

**Layer 2: Ad set (Audience + Budget)**

## 4. TARGETING STRATEGY

**Audience 1: Lookalike**
- Create from top 1% engaged followers
- Cost: $0.15-0.40 (cheapest)
- Result: 2-3x cheaper than cold

**Audience 2: Interest**
- Related creators + communities
- Cost: $0.30-0.70
- Result: Warm audience, decent intent

**Audience 3: Retargeting**
- Website visitors ($0.10-0.30)
- Email list
- Cost: Warmest, cheapest
- Result: Highest conversion

## 6. BEST PRACTICES

**Creative specs:**
- Vertical video (9:16) best
- Captions in video (sound off default)
- Fast hook (first 3 sec)
- 30-60 seconds
- Personal = highest engagement

**Targeting:**
- Start broad (18-65, all genders)
- Test lookalike first
- Exclude current followers
- Separate retargeting

**Bidding:**
- Automatic bidding (platform optimizes)
- Start $1-2/day
